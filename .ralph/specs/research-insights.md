# 핵심 인사이트 요약

> Research Phase 2 | 2026-02-07  
> PM 의사결정용 권고사항

---

## Executive Summary

**한 줄 요약**: 한국 사무직 AI 도구 시장은 "보안 때문에 못 쓰는" 수요가 넘치지만, 이를 충족하는 로컬 도구가 전무함. OpenKlaw는 **HWP + 한국어 + 로컬**이라는 유일한 조합으로 블루오션 진입 가능.

---

## 1. 타겟 사용자 Pain Point TOP 3

### 🥇 #1: "AI 쓰고 싶은데 회사에서 막아요"

**현상**
- 대기업 90%+, 공공기관 100% ChatGPT/Claude 차단
- 보안팀 정책: "외부 서버에 데이터 전송 금지"
- 결과: 개인 핸드폰/집에서 몰래 사용 → 리스크

**수치**
- 예상 잠재 사용자: 한국 사무직 500만명 중 최소 200만명
- 기업 AI 접근 가능자: 10% 미만

**OpenKlaw 해결책**
```
✅ 100% 로컬 실행
✅ 네트워크 요청 Zero
✅ IT 부서 승인 논리: "데이터가 나가지 않습니다"
```

---

### 🥈 #2: "HWP 다루는 AI가 없어요"

**현상**
- 정부기관: HWP 100% 사용
- 대기업: HWP 60-80% (삼성, LG, 현대 등)
- 글로벌 AI 도구: HWP 지원 0%

**수치**
- HWP 사용 기업/기관: 약 30만개 추정
- 일일 HWP 문서 생성량: 수백만 건

**OpenKlaw 해결책**
```
✅ pyhwp 통합으로 HWP 파싱
✅ HWP 문서 요약/분석
✅ HWP → 텍스트 추출 → LLM 처리
```

---

### 🥉 #3: "반복 작업에 시간 너무 많이 써요"

**현상**
- 주간/월간 보고서: 매번 비슷한 포맷
- 이메일: 격식 갖추기에 시간 낭비
- 회의록: 회의 시간 = 회의록 작성 시간

**시간 낭비 (주당)**
| 업무 | 평균 시간 | AI 절감 가능 |
|------|----------|-------------|
| 보고서 | 8시간 | 4시간 (50%) |
| 이메일 | 6시간 | 3시간 (50%) |
| 회의록 | 4시간 | 3시간 (75%) |
| **합계** | 18시간 | 10시간 |

**OpenKlaw 해결책**
```
✅ 이메일 초안 자동 생성
✅ 보고서 템플릿 작성
✅ 회의 내용 → 회의록 정리
```

|------|-----------|-----------|

### 2.3 "Wow" 시나리오

```
1. 설치 (3분)
   - Windows: 설치파일 더블클릭
   - Mac: `npm install -g openklaw`
   
2. 첫 실행 (2분)
   - 모델 자동 다운로드 (Qwen2.5-3B)
   - 한국어 환영 메시지
   
3. 첫 사용 (즉시)
   - "이 HWP 파일 3문장으로 요약해줘"
   → 드래그앤드롭
   → 10초 후 요약 완성
   
4. 반응
   - "와, 이게 내 노트북에서 돌아가는 거야?"
   - "회사에서 써도 되겠네!"
```

## 3. 초기 진입 전략

### 3.1 타겟 순서

```
Phase 1: 개발자/파워유저 (0-3개월)
├── 채널: 긱뉴스, 디스커드, GitHub
├── 목표: 1,000 설치
└── 메시지: "드디어 HWP 지원하는 로컬 AI"

Phase 2: 직장인 얼리어답터 (3-6개월)
├── 채널: 블라인드, 유튜브
├── 목표: 10,000 설치
└── 메시지: "회사에서 맘껏 쓸 수 있는 AI"

Phase 3: 일반 직장인 (6-12개월)
├── 채널: 네이버 블로그, 카페
├── 목표: 50,000 설치
└── 메시지: "야근 줄여주는 AI 비서"
```

### 3.2 바이럴 포인트

1. **긱뉴스 첫 글**
   - 제목: "HWP 읽는 로컬 AI 만들어봤습니다"
   - 핵심: 기술 스택, 설치 방법, 데모 GIF

2. **블라인드 자랑글**
   - 제목: "회사 노트북에서 AI 쓰는 법 (합법)"
   - 핵심: Before/After 시간 비교

3. **유튜브 리뷰**
   - 제목: "ChatGPT 막힌 회사 직원 필수앱"
   - 핵심: 설치 과정, 실사용 시연

### 3.3 경쟁 회피

| 경쟁 상황 | 대응 |
|----------|------|
| "Ollama 쓰면 되잖아" | "CLI 아시는 분은 Ollama가 좋죠. 저희는 클릭 한 번으로 HWP 요약되는 게 목표입니다" |
| "Jan.ai 있는데" | "Jan.ai 훌륭하죠. 다만 HWP 지원이랑 한국어 UI는 저희가 유일합니다" |
| "회사 AI 쓰면 되잖아" | "접근권 있으신 분은 그게 낫죠. 없는 분들을 위한 거예요" |
